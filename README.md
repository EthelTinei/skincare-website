# Клиентская часть веб-приложения для создания персонализированного графика ухода за кожей

## Описание
Веб-приложение позволяет пользователям формировать индивидуальный график ухода за кожей на основе типа кожи, возраста и проблем. Реализована клиентская часть с использованием современных стандартов фронтенд-разработки.

## Стек технологий
- HTML5, CSS3 (БЭМ, Grid/Flexbox, адаптивная вёрстка)
- JavaScript ES6+ (ООП, классы, модули, async/await)
- Vite (сборка, dev-сервер)
- Git + GitHub Pages (деплой)

## Функциональность
- Настройка профиля кожи (тип, возраст, проблемы)
- Персонализированные рекомендации через mock-API
- Двухколоночный график (Утро/Вечер) с сохранением в localStorage
- Тёмная/светлая тема с сохранением предпочтений
- Валидация форм, обработка ошибок, toast-уведомления
- Полная адаптивность (mobile-first)

## Запуск проекта
npm install
npm run dev

## Сборка и деплой
npm run build
